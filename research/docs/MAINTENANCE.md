# Maintenance Schedule

## Daily (2–3 minutes)
- [ ] Check water level in reservoirs (top off if low)
- [ ] Check pH (target 5.8–6.2)
- [ ] Quick visual inspection — look for wilting, discoloration, pests
- [ ] Verify lights turned on/off correctly

## Weekly (10–15 minutes)
- [ ] Measure EC/TDS (leafy greens: 600–900 PPM / fruiting: 1,000–1,500 PPM)
- [ ] Clean any visible algae
- [ ] Inspect roots (healthy = white and firm / brown + slimy = root rot)
- [ ] Check air pump is bubbling properly
- [ ] Harvest mature crops

## Every 2 Weeks (20–30 minutes)
- [ ] Complete nutrient solution change (drain, clean reservoir, refill)
- [ ] Deep clean reservoir walls
- [ ] Check for root tangles and trim if needed
- [ ] Inspect all tubing connections

## Monthly (30 minutes)
- [ ] Calibrate pH and EC meters
- [ ] Inspect air stones (replace if clogged)
- [ ] Check light height and adjust for plant growth
- [ ] Clean fan blades
- [ ] Inventory nutrients and supplies — reorder if low

## Troubleshooting Quick Reference

| Symptom | Likely Cause | Fix |
|---------|-------------|-----|
| Yellow leaves (lower) | Nitrogen deficiency | Increase nutrients / check pH |
| Brown leaf tips | Nutrient burn | Dilute solution / add plain water |
| Wilting despite water | Root rot (Pythium) | Check root health, lower water temp below 75°F |
| Slow growth | pH out of range | Adjust to 5.8–6.2 |
| Algae growth | Light hitting reservoir | Cover all light leaks, use opaque containers |
| Fungus gnats | Wet growing medium surface | Keep top layer of LECA dry, use yellow sticky traps |
| Stretchy/leggy plants | Insufficient light | Lower light or increase wattage |

## pH Quick Guide
- **Add nutrients FIRST, then adjust pH** (nutrients change pH)
- Target: 5.8–6.2 for most crops
- Use pH Down (phosphoric acid) more than pH Up
- Small adjustments — a few drops at a time
- Let tap water sit 24–48 hours to off-gas chlorine before use

## Nutrient Mixing (GH Flora Trio)
For leafy greens per gallon:
- FloraMicro: 2.5 mL
- FloraGro: 2.5 mL
- FloraBloom: 2.5 mL

For fruiting plants (peppers) per gallon:
- FloraMicro: 5 mL
- FloraGro: 2.5 mL
- FloraBloom: 5 mL

Always add FloraMicro to water FIRST, stir, then add the others.
